# 💰 Money Game App

A finance simulation web application built by **Ruturaj Morey**.  
This project allows users to learn investing concepts by simulating real-world financial decisions like trading stocks, managing portfolios, and competing on a leaderboard.

## 📌 About the Project

Money Game is designed to simulate a real-world financial environment where users start with virtual money and grow their wealth through smart investment strategies.

The application focuses on:
- Learning financial concepts
- Practicing investment strategies
- Building interactive and real-time systems

---

## 🎯 Features

### 📊 Trading System
- Buy and sell stocks
- Dynamic price updates
- Market trend simulation

### 💼 Portfolio Management
- Track owned stocks
- View total investment value
- Monitor profits and losses

### 🏆 Leaderboard
- Compete with other users
- Ranking based on balance/net worth

### 📈 Dashboard
- Visual charts using Chart.js
- Net worth tracking
- Performance insights

### 🤖 AI & Simulation (Advanced)
- Simulated market trends
- Random events and price changes
- AI-based investment ideas (future scope)
